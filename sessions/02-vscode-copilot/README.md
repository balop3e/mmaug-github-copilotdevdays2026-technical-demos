# Session 2 - VS Code + GitHub Copilot

## Goal

Use Copilot inside VS Code to redesign and improve a static web page.

## Folder contents

- starter/web/ - starter HTML, CSS, and JS
- challenges/ - prompts and implementation tasks

## Local run

```bash
cd sessions/02-vscode-copilot/starter/web
python -m http.server 8080
```

Open http://127.0.0.1:8080.

## Deploy options

Option A - GitHub Pages:

1. Push this repository to GitHub.
2. In repository settings, enable Pages from root.
3. Share the Pages URL.

Option B - Any static host:

Upload files in starter/web to Netlify, Vercel, or Azure Static Web Apps.
