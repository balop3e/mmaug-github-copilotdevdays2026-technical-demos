# Challenge 01 - Design-First Iteration

## Task

Use Copilot in VS Code to redesign the page without changing the data model.

## Acceptance criteria

- Improve visual hierarchy and spacing
- Add focus styles and keyboard accessibility
- Add one loading or reveal animation
- Keep mobile layout usable

## Suggested prompts

- Propose a bold visual direction and update CSS variables accordingly.
- Improve accessibility with semantic structure and focus states.
- Add lightweight entrance animation for list items.
