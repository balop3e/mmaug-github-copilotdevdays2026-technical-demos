# Challenge 01 - Safer Command Automation

## Task

Use Copilot CLI to generate a script that archives completed tasks from tasks.csv.

## Acceptance criteria

- Creates archive.csv with completed rows
- Removes archived rows from original file
- Supports dry-run mode
- Includes clear usage output

## Suggested prompts

- Create a safe bash script to archive done rows in this CSV.
- Add --dry-run and --help support.
