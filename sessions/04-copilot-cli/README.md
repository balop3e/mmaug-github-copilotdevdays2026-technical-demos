# Session 4 - GitHub Copilot CLI

## Goal

Practice intent-to-command workflows and safe terminal automation patterns.

## Folder contents

- starter/ - runnable shell scripts and sample data
- challenges/ - CLI challenge tasks

## Local run

```bash
cd sessions/04-copilot-cli/starter
chmod +x *.sh
./setup-demo-data.sh
./analyze-demo-data.sh
```

## Deploy options

Option A - Codespaces:

Run scripts directly in Codespaces terminal.

Option B - Workshop VM:

Copy this folder to a Linux/macOS VM and run the scripts.
