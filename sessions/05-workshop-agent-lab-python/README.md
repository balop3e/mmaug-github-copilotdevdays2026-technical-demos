# Session 5 - 1 Hour Workshop: Agent Lab Python

## Goal

Run the official Agent Lab Python workshop in a guided and self-paced format.

## References

- Workshop site: https://copilot-dev-days.github.io/agent-lab-python/
- Source repo: https://github.com/copilot-dev-days/agent-lab-python

## Workshop timing

1. Setup and Context Engineering - 15 min
2. Design-First Frontend - 15 min
3. Custom Quiz Master - 10 min
4. Multi-Agent Development - 20 min

## Local run

```bash
./bootstrap-agent-lab-python.sh
```

## Deploy options

Option A - Codespaces:

Run the workshop in a cloud development environment and share links with participants.

Option B - Speaker machine:

Pre-clone the workshop repository and provide the prepared branch to attendees.
