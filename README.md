# MMAUG GitHub Copilot Dev Days 2026 Technical Demos

This repository contains all technical demos for GitHub Copilot Dev Days Malta 2026.

## Repository structure

- sessions/01-every-workflow
- sessions/02-vscode-copilot
- sessions/03-agentic-workflows
- sessions/04-copilot-cli
- sessions/05-workshop-agent-lab-python

## Quick start

1. Clone this repository.
2. Choose a session folder.
3. Follow the session README for setup, run, and deployment.

## Event-day self-paced flow

1. Start with Session 2 (VS Code + Copilot).
2. Continue with Session 4 (Copilot CLI).
3. Complete Session 3 (Agentic Workflows).
4. Finish the 1-hour workshop in Session 5.

## Participant submissions

- Open Issues and select the session challenge submission template.
- Submit your repository, evidence, and implementation summary.

- GitHub Actions workflow validates each session starter on push and pull request.
